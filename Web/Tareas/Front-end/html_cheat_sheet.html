HTML Cheat Sheet

1. a:

Explicación:

El tag "a" represta "Anchor". Es utilizado para crear un hyperlink ya sea a otra página web (con paths absolutos) o a otra sección en la página web (con paths relativos).

Ejemplo:

Mi repositorio personal en Github


2. p:

Explicación:

El tag "p" represta "Paragraph". Es un elemento semántico cuyo contenido representa un párrafo.

Ejemplo:

Este es un párrafo dentro de mi página web.


3. h1, h2, ... h6:

Explicación:

Los tags "h1, h2, ... h6" represtan "Headings". Son elementos a nivel de bloque que representan encabezados
para segmentar el contenido y establecer jerarquías dentro del contenido de la página web.

Ejemplo:

h1

h2

h3

h4

h5
h6


4. div:

Explicación:

Un div es un elemento a nivel de bloque que se suele utilizar para identificar grandes agrupaciones de contenido
y que ayuda a construir la disposición y el diseño de una página web. En un elemento sin valor semántico.

Ejemplo:

Este es un ejemplo del uso de un div en mi página web.


5. span:

Explicación:

Un span es un elemento de nivel inline utilizado habitualmente para identificar agrupaciones más pequeñas de texto
dentro de un elemento de nivel bloque. En un elemento sin valor semántico.

Ejemplo:

Mi color favorito es el rojo.


6. strong:

Explicación:

El tag strong funciona para establecer gran importancia en un texto. Es un elemento con valor semántico.

Ejemplo:

Importante: No habrán clases el día de mañana.


7. i:

Explicación:

El tag i se utiliza semánticamente para transmitir el texto en una voz o tono alternativo.

Ejemplo:

El nombre Paúl se pronuncia "pa-ú-l", no "pol".


8. u:

Explicación:

El tag u, que significa "Underline", se utiliza semánticamente para subrayar un texto para llamar la atención sobre él.

Ejemplo:

El título del libro es Dune.


9. b:

Explicación:

El tag b El elemento b significa semánticamente desplazar estilísticamente el texto.

Ejemplo:

Importante: No habrán clases el día de mañana.


10. strike:

Explicación:

El elemento strike en HTML es utilizado para mostrar texto que ha sido tachado,
indicando que el texto es incorrecto o ya no es relevante.

Ejemplo:

El precio original era $199.99 $149.99.


11. small:

Explicación:

El elemento small se utiliza para mostrar texto en un tamaño de fuente más pequeño,
indicando información secundaria o de menor importancia, como avisos legales o notas al pie.

Ejemplo:

Precio del producto: $50.00 (sin incluir impuestos)


12. em:

Explicación:

El elemento em en HTML se utiliza para enfatizar texto, lo cual suele resultar en que el texto se muestre en cursiva.
Esto indica que el texto tiene un énfasis o importancia especial dentro del contexto.

Ejemplo:

Para lograr un éxito significativo, es necesario un compromiso total.


13. header:

Explicación:

El elemento header se utiliza para identificar la parte superior de una página, artículo, sección u otro segmento de una página.

Ejemplo:

Nombre de la compañía

Introductory text


14. section:

Explicación:

El elemento section en HTML se utiliza para definir una sección dentro de un documento,
la cual agrupa contenido temáticamente relacionado. Suele estar acompañado de un encabezado
(como h1 a h6) y organiza el contenido de la página en segmentos lógicos, facilitando tanto la navegación como la accesibilidad del documento.

Ejemplo:

Nuestros Servicios

Ofrecemos una amplia variedad de servicios para satisfacer tus necesidades, incluyendo desarrollo web, marketing digital, y diseño gráfico.

  • Desarrollo Web
  • Marketing Digital
  • Diseño Gráfico


15. article:

Explicación:

El elemento artículo se utiliza para identificar una sección de contenido independiente y autónomo que puede distribuirse o reutilizarse de
forma independiente.

Ejemplo:

Nombre de la compañía

Introductory text


16. aside:

Explicación:

El elemento aside contiene cosas como explicaciones breves o barras laterales que están relacionadas con la información que lo rodea.

Ejemplo:

Nombre de la compañía

Introductory text

Otra información relevante


17. footer:

Explicación:

El elemento footer cidentifica el cierre de alguna sección de una página web.

Ejemplo:

Introductory text

Fin de artículo

23. table:

Explicación:

El elemento table se utiliza para crear tablas en la página web.
Las tablas se componen de varios elementos que permiten definir su estructura y contenido, como filas (tr), celdas de encabezado (th) y celdas de cuerpo (td).

Ejemplo:

Maria López marialopez@example.com 35


24. ul:

Explicación:

El elemento ul se utiliza para hacer unordered lists.

Ejemplo:

  • Leche
  • Pan
  • Huevos


25. ol:

Explicación:

El elemento ul se utiliza para hacer ordered lists.

Ejemplo:

Lista del súper:

  1. Leche
  2. Pan
  3. Huevos
